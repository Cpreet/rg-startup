# Introduction to javascript

Javascript is a simple scripting language.
A lot of runtimes exist for excuting the language in various contexts including

- Modern web browsers
- Server systems
- Shells
- Some gaming platforms

* Due to its versatility, it has wide adoption and a lot of support in communities.

* The original language was developed by 1995 **Brenden Eich** at NetScape.

* It was named after the then famous **java** language but is considerably different.

* The modern standard of javascript is maintained by the **European Computer Manudacturers Association** (abbv. ECMA) and is offically dubbed ECMAScript. The standard is **ECMA-262 specification**.

* The **TC39** group is a voluntary commity that keeps the standard upto date.

* The ECMAScript standard verisons are as follows
