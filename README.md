# Fullstack development notes

- This respository contains the notes for the fullstack development course

The notes structures are as follows

```sh
.
├─/html
├─/css
├─/javascript
├─/git_github
└─/cmd_bash
```

The named folder will contain the notes of the topic and will be linked with the other folders.

The notes are displayed in markdown format such that they can be viewed normally as well as in Markdown renderers.
